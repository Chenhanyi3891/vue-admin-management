# 通用后台管理系统-前端 vue-admin-management

### 项目介绍：

1、实现前端通用后台管理页面，包含侧边导航栏、用户登录、页面路由控制、面包屑、数据图表展示等功能
2、路由控制、权限管理

### 技术栈

开发语言：HTML、CSS、Javascrip
框架：Vue、vue-cli 脚手架、Vue-Router 路由
组件：Element-UI、Axios、Echart
